# TokoFashion - Landing Page

Landing page untuk toko fashion online menggunakan Tailwind CSS v4.

## Tech Stack

- HTML5
- Tailwind CSS v4
- Flowbite (untuk komponen interaktif)
- Font: Satoshi

## Cara Menjalankan

1. Install dependencies:
   ```bash
   npm install
   ```

2. Compile Tailwind CSS (development):
   ```bash
   npx @tailwindcss/cli -i ./src/input.css -o ./src/output.css --watch
   ```

3. Buka file `src/index.html` di browser

## Cara Build untuk Production

```bash
npx @tailwindcss/cli -i ./src/input.css -o ./src/output.css --minify
```

## Fitur

- Responsive design (Mobile, Tablet, Desktop)
- Top navigation bar
- Navbar dengan dropdown menu
- Hero section
- Category section
- Product cards
- Subscribe form
- FAQ accordion
- Footer dengan social media links
